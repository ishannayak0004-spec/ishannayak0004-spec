# 🔍 Anomaly Detection using Autoencoder (TEP Dataset)

</summary>

---

## 📖 Overview

Designed and implemented a deep learning-based anomaly detection system using an **Autoencoder Neural Network** trained on the **Tennessee Eastman Process (TEP)** dataset.

The model learns the normal behavior of an industrial process and identifies abnormal conditions by measuring reconstruction error.

---

## 🎯 Objectives

✔ Detect abnormal industrial process behavior

✔ Learn latent feature representations

✔ Reduce false alarms

✔ Improve predictive maintenance

---

## 🏗 Architecture

```text
Industrial Sensor Data
          │
          ▼
   Data Preprocessing
          │
          ▼
      Autoencoder
   (Encoder → Decoder)
          │
          ▼
 Reconstruction Error
          │
          ▼
Normal / Anomaly Prediction
```
